d4-rails
========

Integrate d4 DSL (for d3 library) with your rails application

## Pre-requisites:

For Rails 3.2+

## Installling

Install the gem by

    gem install d4-rails

OR

Include the gem in your Gemfile and run bundle install.

    gem 'd4-rails'

Also require `d4.js` is your `application.js`

    //= require 'd4'

That's it! :pray:
